# Travel Guide Application

Travel application where you can search for hotels by country

## What did We use ?

- [x] MVVM architecture
- [x] Dagger Hilt
- [x] Retrofit
- [x] Coroutine
- [x] Glide
- [x] View Binding
- [x] Navigation
- [x] View Pager (Onboarding), Recycler View
